Shortcode para incluir botón que ejecute widget de \"hablamos\"

== Description ==
Este plugin te permite agregar un botón en tu sitio web que ejecuta el widget de \"hablamos\" de <a href="www.fonvirtual.com">Fonvirtual</a>. Con este botón podrás ofrecer a tus visitantes una forma rápida y fácil de hablar con un representante de atención al cliente.

Características
1. Shortcode para incluir el botón en cualquier lugar de tu web
2. Personalización de la apariencia del botón con CSS
3. Integración con Google Analytics para seguimiento de eventos de clic
3. Compatibilidad con todas las versiones de WordPress

== Installation ==
1. Instala y activa el plugin en tu sitio WordPress.
2. Agrega el shortcode [boton_hablamos] en la página o post donde deseas incluir el botón.
3. Personaliza el texto del botón y otros atributos según tus necesidades.
4. Guarda y publica tus cambios.
